# Frontend (Vue 3 + Vite)

SPA com login/cadastro, listagem e criação de despesas, incluindo interceptors Axios, guards de rota e validação de formulários.

## Setup

```bash
cp .env.example .env
npm install
npm run dev
```

## Variável de ambiente

```bash
VITE_API_BASE_URL=http://localhost:8000/api
```

## Testes

```bash
npm run test:unit
npm run test:e2e
```
